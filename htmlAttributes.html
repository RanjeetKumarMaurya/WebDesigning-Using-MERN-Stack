HTML Attributes

Html Attributes helps defining the behaviour of HTML elements.
All the HTML Attributes have two property

  1. Attribute Name: It is a unique identifier that specified type of info being provided. Like id, src etc.
  2. Attribute Value: It shows the actual data associated with attribute name.

Going forward we will use all the attributes, for the time being we will learn some important HTML attributes.

  • HTML src Attribute: Allows loading image/videos/audio from a url

  • HTML alt Attribute: Used to show something if primary content failed to load and display the value assigned to it.

  • HTML width and height Attribute: Used to adjust the width and height of image in pixels.

  • HTML id Attribute: Provides unique identification of an element. Help in case if two element have same name.

  • HTML href Attribute: The href attribute is used to specify a link to any address. This attribute is used along with the < a > tag.
